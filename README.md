# Development

## 1. Install dependencies

```sh
cd backend
pip install -r requirements.txt
```
And in frontend:
```sh
cd frontend
npm install
```
## 2. Start the server

```sh
cd backend
py -m main
```
And in frontend:
```sh
cd frontend
npm run serve
```
